# Ayush Chhetri - Portfolio Website

This is my personal portfolio website. It shows my education, skills, certificates, projects, and contact details.

## What I used

- HTML
- CSS
- JavaScript
- React (used only for a small skills filter, loaded through a CDN, no installation needed)

## Files in this project

- index.html - main page
- main.css - all the styling
- script.js - handles animations and nav highlighting
- skills-filter.jsx - React component for filtering skills
- Ayush_Chhetri_Resume.pdf - my resume, downloadable from the site

## How to run it

Just open index.html in your browser. No installation or setup needed.
